# Pokédex 
Projeto de uma Pokédex interativa usando HTML, CSS, JavaScript e consumo da PokéAPI.

## Funcionalidades
- Lista os 20 primeiros Pokémon com nome e imagem.
- Visual responsivo com efeito hover nos cards.
- Consumo da API REST PokéAPI.

## Tecnologias
- HTML5
- CSS3
- JavaScript ES6
- Git + GitHub
